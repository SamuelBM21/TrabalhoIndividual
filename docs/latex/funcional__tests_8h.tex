\doxysubsection{Detailed Description}
Declaração de fluxos de teste e protótipos de testes funcionais. 

Este arquivo contém implementações simples de subclasses de \mbox{\hyperlink{classFlow}{Flow}} usadas exclusivamente para testes funcionais do simulador (\mbox{\hyperlink{classExponentialFlow}{Exponential\+Flow}} e logistic\+Flow), bem como os protótipos das funções de teste que verificam cenários comportamentais do sistema/model.

As classes aqui definidas são utilitárias para os testes e não fazem parte da API pública do simulador. Elas são usadas para validar a integração entre \mbox{\hyperlink{classModel}{Model}}, \mbox{\hyperlink{classSystem}{System}} e \mbox{\hyperlink{classFlow}{Flow}} durante execuções de teste automatizadas.
