\doxysubsection{Detailed Description}
Implementação parcial da classe \mbox{\hyperlink{classFlow}{Flow}}. 

Esta classe implementa a maior parte da interface \mbox{\hyperlink{classFlow}{Flow}}, incluindo\+:
\begin{DoxyItemize}
\item Armazenamento dos ponteiros source e target;
\item Construtores;
\item Operador de atribuição;
\item Métodos de acesso set/get.
\end{DoxyItemize}

O método \mbox{\hyperlink{classFlowImpl_a88d14f759988f1dcf393b83a93aea1f1}{execute()}} permanece abstrato, pois cada fluxo concreto define sua própria equação.
