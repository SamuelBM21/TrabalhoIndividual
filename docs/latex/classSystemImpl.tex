\doxysubsection{Detailed Description}
Implementação concreta da classe \mbox{\hyperlink{classSystem}{System}}. 

Armazena um único valor numérico, representando o estado atual de um elemento do modelo. A classe oferece métodos para definição, obtenção e cópia desse valor.
