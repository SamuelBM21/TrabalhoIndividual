\doxysubsection{Detailed Description}
Implementação concreta da interface \mbox{\hyperlink{classModel}{Model}}. 

A classe \mbox{\hyperlink{classModelImpl}{Model\+Impl}} fornece uma implementação padrão do mecanismo de simulação, armazenando listas de Systems e Flows e aplicando os fluxos ao longo do tempo.

Esta classe é responsável por\+:
\begin{DoxyItemize}
\item Gerenciar a memória e organização interna dos objetos do modelo;
\item Fornecer iteradores constantes para acesso às coleções;
\item Executar a simulação conforme a lógica definida nos Flows.
\end{DoxyItemize}
