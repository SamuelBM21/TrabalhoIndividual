\doxysubsection{Detailed Description}
Declaração da classe abstrata \mbox{\hyperlink{classFlow}{Flow}}, que modela um fluxo entre dois Systems. 

A classe \mbox{\hyperlink{classFlow}{Flow}} representa uma transferência de valor entre um sistema de origem (source) e um sistema de destino (target). Ela é a classe base para todas as implementações concretas de fluxos do simulador (por exemplo\+: fluxo linear, fluxo logístico, etc.). Subclasses devem implementar o método virtual puro execute(), que calcula o valor do fluxo em um determinado passo de simulação.
