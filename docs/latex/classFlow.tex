\doxysubsection{Detailed Description}
Interface abstrata que define um fluxo entre dois Systems. 

Cada \mbox{\hyperlink{classFlow}{Flow}} conecta dois Systems por meio de ponteiros — source (origem) e target (destino). O \mbox{\hyperlink{classModel}{Model}} executa todos os fluxos a cada iteração para calcular como os valores dos Systems devem evoluir.

Subclasses concretas devem implementar o método \mbox{\hyperlink{classFlow_a619be0b590c78202127bc6ac7fb04029}{execute()}}, que define a regra do fluxo.
