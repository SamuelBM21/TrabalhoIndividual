\doxysubsection{Detailed Description}
Classe base abstrata que define um fluxo entre dois Systems. 

Cada \mbox{\hyperlink{classFlow}{Flow}} contém ponteiros para um \mbox{\hyperlink{classSystem}{System}} de origem e um \mbox{\hyperlink{classSystem}{System}} de destino. O \mbox{\hyperlink{classModel}{Model}} utiliza objetos \mbox{\hyperlink{classFlow}{Flow}} para computar e aplicar transferências entre Systems em cada iteração da simulação.
