\doxysubsection{Detailed Description}
Implementação base concreta da interface \mbox{\hyperlink{classFlow}{Flow}}. 

\mbox{\hyperlink{classFlowImpl}{Flow\+Impl}} fornece uma implementação padrão para o armazenamento e manipulação dos ponteiros {\ttfamily source} e {\ttfamily target}, deixando apenas o método execute() como abstrato, pois sua definição depende do tipo de fluxo.

Essa classe pode ser usada diretamente como superclasse para fluxos customizados, evitando repetição de código.
