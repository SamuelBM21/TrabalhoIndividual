\doxysubsection{Detailed Description}
Arquivo principal responsável por executar os testes funcionais do simulador. 

Este arquivo contém apenas a função {\ttfamily \mbox{\hyperlink{funcional_2main_8cpp_a840291bc02cba5474a4cb46a9b9566fe}{main()}}}, cuja função é chamar os testes funcionais implementados em {\ttfamily unit\+\_\+\+Classe.\+cpp}. Ele serve como ponto de entrada para validar o comportamento de cada método das classes \mbox{\hyperlink{classModel}{Model}}, \mbox{\hyperlink{classSystem}{System}} e \mbox{\hyperlink{classFlow}{Flow}}.
