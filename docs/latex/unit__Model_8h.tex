\doxysubsection{Detailed Description}
Declaração dos testes unitários para a classe \mbox{\hyperlink{classModel}{Model}} e \mbox{\hyperlink{classModelImpl}{Model\+Impl}}. 

Este arquivo contém as assinaturas das funções responsáveis por testar o funcionamento interno da classe \mbox{\hyperlink{classModelImpl}{Model\+Impl}}, incluindo\+:
\begin{DoxyItemize}
\item Construtor e destrutor;
\item Métodos de gerenciamento de Systems e Flows (add/remove);
\item Métodos de iteração (begin, end, size);
\item Execução da simulação (run);
\item Validação do operador de atribuição (=).
\end{DoxyItemize}

Cada função implementa um teste isolado, garantindo que \mbox{\hyperlink{classModelImpl}{Model\+Impl}} mantenha o controle correto de seus componentes e funcione de forma consistente ao longo da execução da simulação.

Os testes são implementados no arquivo \mbox{\hyperlink{unit__Model_8cpp}{unit\+\_\+\+Model.\+cpp}}.
