\doxysubsection{Detailed Description}
Interface abstrata para representação e execução de modelos na simulação. 

A classe \mbox{\hyperlink{classModel}{Model}} define a estrutura básica para implementação de modelos baseados na Teoria Geral de Sistemas (TGS). ~\newline


Um \mbox{\hyperlink{classModel}{Model}} é responsável por\+:
\begin{DoxyItemize}
\item Gerenciar coleções de objetos \mbox{\hyperlink{classSystem}{System}} (entidades que armazenam valores).
\item Gerenciar coleções de objetos \mbox{\hyperlink{classFlow}{Flow}} (relações que transferem valores entre systems).
\item Coordenar o processo de simulação ao longo do tempo.
\end{DoxyItemize}

Esta classe funciona como o núcleo lógico da simulação, servindo de base para implementações concretas como \mbox{\hyperlink{classModelImpl}{Model\+Impl}}.
