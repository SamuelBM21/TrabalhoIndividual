\doxysubsection{Detailed Description}
Declaração da classe \mbox{\hyperlink{classModel}{Model}}, responsável pelo gerenciamento da simulação. 

A classe \mbox{\hyperlink{classModel}{Model}} é o núcleo do simulador baseado na Teoria Geral de Sistemas. Ela armazena e controla coleções de objetos do tipo \mbox{\hyperlink{classSystem}{System}} e \mbox{\hyperlink{classFlow}{Flow}}, executando a evolução temporal do sistema através da aplicação dos fluxos sobre os sistemas.

O \mbox{\hyperlink{classModel}{Model}} funciona como um \char`\"{}ambiente de execução\char`\"{}, coordenando\+:
\begin{DoxyItemize}
\item A lista de sistemas presentes no modelo
\item A lista de fluxos conectando esses sistemas
\item O processo de execução (simulação) no tempo
\end{DoxyItemize}
