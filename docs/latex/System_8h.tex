\doxysubsection{Detailed Description}
Interface abstrata que representa um elemento de armazenamento do modelo. 

A classe \mbox{\hyperlink{classSystem}{System}} modela um \char`\"{}estoque\char`\"{} (stock) dentro da Teoria Geral de Sistemas. Cada \mbox{\hyperlink{classSystem}{System}} contém um valor numérico que pode ser modificado ao longo da simulação por meio de objetos \mbox{\hyperlink{classFlow}{Flow}}.

Esta classe define a API básica para manipulação de sistemas e deve ser implementada por classes concretas como \mbox{\hyperlink{classSystemImpl}{System\+Impl}}.
