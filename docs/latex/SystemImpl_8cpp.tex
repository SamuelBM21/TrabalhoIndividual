Este arquivo contém a implementação da classe \mbox{\hyperlink{classSystemImpl}{System\+Impl}}, responsável por armazenar e manipular o valor interno de um \mbox{\hyperlink{classSystem}{System}} no simulador.

\mbox{\hyperlink{classSystemImpl}{System\+Impl}} fornece\+:
\begin{DoxyItemize}

\item Operador de atribuição;
\item Métodos set\+Value() e get\+Value();
\item Regras básicas de consistência e encapsulamento.
\end{DoxyItemize}

Esta classe é utilizada diretamente pelo \mbox{\hyperlink{classModel}{Model}} e pelos Flows para compor a dinâmica dos estoques durante a simulação baseada na Teoria Geral de Sistemas.
