\doxysubsection{Detailed Description}
Classe responsável por gerenciar sistemas, fluxos e a simulação completa. 

O \mbox{\hyperlink{classModel}{Model}} armazena dois conjuntos principais\+:
\begin{DoxyItemize}
\item Uma lista de objetos \mbox{\hyperlink{classSystem}{System}}, representando entidades com um valor armazenado.
\item Uma lista de objetos \mbox{\hyperlink{classFlow}{Flow}}, que representem operações de transferência de valor entre Systems.
\end{DoxyItemize}

A função principal do \mbox{\hyperlink{classModel}{Model}} é executar a simulação entre um tempo inicial e final, aplicando todos os fluxos existentes a cada passo.

\doxysubsubsection{\texorpdfstring{run()}{run()}}
{\footnotesize\ttfamily bool Model\+::run (\begin{DoxyParamCaption}\item[{int}]{start\+Time,  }\item[{int}]{end\+Time }\end{DoxyParamCaption})}



Executa a simulação completa. 

Executa a simulação do modelo entre dois instantes de tempo.

A simulação ocorre aplicando todos os fluxos do modelo entre as iterações de tempo fornecidas. O comportamento detalhado depende da implementação de cada tipo concreto de \mbox{\hyperlink{classFlow}{Flow}}.


\begin{DoxyParams}{Parameters}
{\em start\+Time} & Tempo inicial da simulação. \\
\hline
{\em end\+Time} & Tempo final da simulação. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
true se a simulação ocorreu normalmente, false caso haja falhas.
\end{DoxyReturn}
Este método percorre o intervalo de tempo definido por {\ttfamily start\+Time} e {\ttfamily end\+Time}, calculando primeiro o valor de todos os fluxos e depois aplicando esses valores aos sistemas associados. O processo ocorre em duas etapas a cada ciclo\+:


\begin{DoxyEnumerate}
\item {\bfseries{Cálculo dos fluxos\+:}} cada \mbox{\hyperlink{classFlow}{Flow}} executa seu método execute(), cujo resultado é armazenado em um vetor temporário. ~\newline
 Isso garante que todos os fluxos sejam calculados antes de alterar qualquer \mbox{\hyperlink{classSystem}{System}}.
\item {\bfseries{Atualização dos sistemas\+:}} para cada fluxo, subtrai-\/se o valor do fluxo do sistema de origem (source) e adiciona-\/se ao sistema de destino (target), caso existam. ~\newline
 Essa abordagem evita dependências indesejadas entre fluxos e garante consistência durante a simulação.
\end{DoxyEnumerate}

\hline
\end{DoxyParams}

true se a simulação foi executada com sucesso. 
\end{DoxyReturn}
