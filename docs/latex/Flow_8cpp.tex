\doxysubsection{Detailed Description}
Implementação da classe \mbox{\hyperlink{classFlow}{Flow}}. 

Implementa a infraestrutura básica para representar um fluxo entre dois sistemas no simulador. Classes derivadas devem sobrescrever execute() para definir o comportamento específico do fluxo.

O \mbox{\hyperlink{classFlow}{Flow}} é utilizado pelo \mbox{\hyperlink{classModel}{Model}} para simular o comportamento dinâmico de um sistema de acordo com a Teoria Geral de Sistemas.
