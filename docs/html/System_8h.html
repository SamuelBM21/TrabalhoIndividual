Detailed Description

Declaração da classe System, que representa um elemento básico do modelo.

A classe System é uma das estruturas fundamentais do simulador baseado na Teoria Geral de Sistemas. Um System representa um "estoque" (stock), isto é, uma entidade que possui um valor armazenado e que pode ser alterado ao longo da simulação por meio dos fluxos (Flows).
