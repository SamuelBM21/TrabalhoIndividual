Detailed Description

Interface abstrata que representa um elemento de armazenamento do modelo.

A classe System modela um "estoque" (stock) dentro da Teoria Geral de Sistemas. Cada System contém um valor numérico que pode ser modificado ao longo da simulação por meio de objetos Flow.

Esta classe define a API básica para manipulação de sistemas e deve ser implementada por classes concretas como SystemImpl.
