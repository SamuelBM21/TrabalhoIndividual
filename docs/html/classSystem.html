Detailed Description

Interface para um elemento de armazenamento dentro do modelo.

Representa uma entidade que possui um único valor numérico, o qual é atualizado conforme os fluxos que incidem sobre ela durante a simulação.

Implementações concretas (como SystemImpl) devem fornecer mecanismos para armazenamento e atualização de valores.
