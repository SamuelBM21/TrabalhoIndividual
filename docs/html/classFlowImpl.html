Detailed Description

Implementação parcial da classe Flow.

Esta classe implementa a maior parte da interface Flow, incluindo:

  • Armazenamento dos ponteiros source e target;
  • Construtores;
  • Operador de atribuição;
  • Métodos de acesso set/get.

O método execute() permanece abstrato, pois cada fluxo concreto define sua própria equação.
