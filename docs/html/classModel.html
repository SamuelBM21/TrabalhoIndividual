Detailed Description

Classe responsável por gerenciar sistemas, fluxos e a simulação completa.

O Model armazena dois conjuntos principais:

  • Uma lista de objetos System, representando entidades com um valor armazenado.
  • Uma lista de objetos Flow, que representem operações de transferência de valor entre Systems.

A função principal do Model é executar a simulação entre um tempo inicial e final, aplicando todos os fluxos existentes a cada passo.

◆ run()

bool Model::run ( int  startTime,
int  endTime 
)

Executa a simulação completa.

Executa a simulação do modelo entre dois instantes de tempo.

A simulação ocorre aplicando todos os fluxos do modelo entre as iterações de tempo fornecidas. O comportamento detalhado depende da implementação de cada tipo concreto de Flow.

Parameters
startTimeTempo inicial da simulação.
endTimeTempo final da simulação.
Returns
true se a simulação ocorreu normalmente, false caso haja falhas.

Este método percorre o intervalo de tempo definido por startTime e endTime, calculando primeiro o valor de todos os fluxos e depois aplicando esses valores aos sistemas associados. O processo ocorre em duas etapas a cada ciclo:

  1. Cálculo dos fluxos: cada Flow executa seu método execute(), cujo resultado é armazenado em um vetor temporário.
    Isso garante que todos os fluxos sejam calculados antes de alterar qualquer System.
  2. Atualização dos sistemas: para cada fluxo, subtrai-se o valor do fluxo do sistema de origem (source) e adiciona-se ao sistema de destino (target), caso existam.
    Essa abordagem evita dependências indesejadas entre fluxos e garante consistência durante a simulação.
Parameters
startTimeTempo inicial da simulação (inteiro).
endTimeTempo final da simulação (inteiro). A simulação ocorre enquanto time < endTime.
Returns
true se a simulação foi executada com sucesso.
