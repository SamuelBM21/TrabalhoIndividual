Detailed Description

Implementação parcial da classe Flow.

Este arquivo contém a implementação da classe FlowImpl, que fornece a infraestrutura básica para representar um fluxo entre dois Systems.

FlowImpl implementa:

  • Construtores (padrão, parametrizado e cópia);
  • Operador de atribuição;
  • Métodos de acesso e modificação de source e target.

O método execute() permanece abstrato, pois cada fluxo concreto deve definir sua própria equação matemática.

FlowImpl é usada como classe base para fluxos específicos utilizados pelo Model durante a simulação.
