Detailed Description

Declaração de fluxos de teste e protótipos de testes funcionais.

Este arquivo contém implementações simples de subclasses de Flow usadas exclusivamente para testes funcionais do simulador (ExponentialFlow e logisticFlow), bem como os protótipos das funções de teste que verificam cenários comportamentais do sistema/model.

As classes aqui definidas são utilitárias para os testes e não fazem parte da API pública do simulador. Elas são usadas para validar a integração entre Model, System e Flow durante execuções de teste automatizadas.
