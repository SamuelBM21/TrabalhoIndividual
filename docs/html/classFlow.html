Detailed Description

Interface abstrata que define um fluxo entre dois Systems.

Cada Flow conecta dois Systems por meio de ponteiros — source (origem) e target (destino). O Model executa todos os fluxos a cada iteração para calcular como os valores dos Systems devem evoluir.

Subclasses concretas devem implementar o método execute(), que define a regra do fluxo.
