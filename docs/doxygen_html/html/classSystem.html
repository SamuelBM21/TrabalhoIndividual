Detailed Description

Representa um elemento de armazenamento dentro do modelo.

Um System contém um único valor numérico, que representa a quantidade atual de um recurso, população, nível, concentração etc., dependendo do contexto do modelo sendo simulado.

Essa classe é manipulada pela classe Model e pelos objetos Flow, que alteram seu valor ao longo da execução da simulação.
